Quiz Game System

Консольная система тестирования на базе Python, использующая архитектурный паттерн разделения логики и данных. Проект демонстрирует работу с коллекциями, обработку типов в реальном времени и нормализацию пользовательского ввода.
Функциональные возможности

    Типизация вопросов: Раздельная логика обработки для текстовых и числовых ответов (text / number).

    Гибкая проверка: Поддержка нескольких вариантов правильных ответов (синонимов) для одного вопроса.

    Нормализация данных: Автоматическая очистка строковых ответов от лишних пробелов и игнорирование регистра (Case-insensitive).

    Масштабируемость: Добавление новых вопросов производится путем расширения списка словарей без изменения исполняемого кода.

Технические особенности
Структура данных (Data Schema)

Вопросы хранятся в списке словарей, где каждый объект имеет строго определенную схему:

    text: Строковое представление вопроса.

    answers: Список (list) допустимых значений.

    type: Флаг типа данных для определения метода обработки.

Алгоритм обработки

Программа использует единый цикл перебора коллекции, где результат каждой итерации (0 или 1) аккумулируется в переменной score. Это избавляет от необходимости использования сложных вложенных условий.
Структура проекта

    quiz.py — функциональное ядро игры и база вопросов.

    README.md — техническое описание системы.

    .gitignore — файл исключений для контроля версий.
